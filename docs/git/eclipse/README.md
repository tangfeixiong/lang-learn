### Tutorial

1. Import as Maven project

Menu: File -> Import

Dialog: Maven -> Existing Maven Projects

选择本地仓库的目录

从Package Explorer视图菜单: Team -> Remote -> Push

从Package Explorer视图菜单: Team -> Merge...
